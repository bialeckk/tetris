# tetris
Tetris game for IPP. (2.2)

Tetris zaprogramowany w języku Python, z wykorzystaniem biblioteki Pygame.
Możliwość przegranej, zwiększenie poziomu trudności, zapisywanie wyników, oprawa graficzna.
